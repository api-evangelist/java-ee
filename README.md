# Java EE (java-ee)
Java EE (Java Platform, Enterprise Edition) was a set of specifications extending the Java SE platform with enterprise features such as distributed computing and web services. In 2017 Oracle transferred Java EE to the Eclipse Foundation, where it was rebranded as Jakarta EE. Jakarta EE 11 continues the platform with profiles for Platform, Web, and Core covering RESTful Web Services, JSON Processing, Persistence, CDI, Servlet, Security, and other enterprise Java specifications.
