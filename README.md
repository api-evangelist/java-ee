# Java EE (java-ee)
Java EE (Java Platform, Enterprise Edition) was a set of specifications extending the Java SE platform with specifications for enterprise features such as distributed computing and web services. Java EE has since been transferred to the Eclipse Foundation and rebranded as Jakarta EE.
